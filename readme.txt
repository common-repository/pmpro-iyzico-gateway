== Description ==
Iyzico paid membership pro is the simple and lightweight implementation of iyzico.com payment service for paid membership only. This plugin support's Turkish Lira Currency of Turkey Country.   

This plugin is paid membership pro extension for payment gateways.

== Installation ==
Installation

Login your Wordpress admin and navigate to Plugins->Add New 

Find the Iyzico Payment Gateway for Paid Membership Pro plugin and activate the module.

After the successful activation of module, navigate to "Memberships->Payment Settings->Payment Gateway select Iyzico Payment Gateway .

Get your api keys from iyzico merchant backend.
